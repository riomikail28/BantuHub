# BantuHub

BantuHub adalah marketplace jasa serba bisa berbasis web responsive dengan rencana pengembangan mobile app.

## Struktur Repository

```text
bantuhub/
├── backend-api/
├── frontend-web/
├── docs/
└── README.md
```

Tahap pertama hanya berfokus pada `backend-api` sebagai Laravel REST API. Frontend belum dibuat.

## Domain Awal

- Role: `admin`, `customer`, `provider`
- Kategori awal:
  - Jasa Rumah
  - Jasa Elektronik
  - Jasa Kendaraan
  - Jasa Kreatif Digital
  - Jasa Pendidikan
  - Jasa Event
  - Jasa Care & Pendampingan Non-Medis
- Fee platform: 2% dari setiap transaksi yang selesai

Booking dan transaksi belum diimplementasikan sampai auth dan database dasar selesai.
